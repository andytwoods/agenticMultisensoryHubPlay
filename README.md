## Project Structure
- `publication-site/`: The Astro + React source code for the interactive website.
- `extracted_content.txt`: Source text extracted from the manuscript.
- `extracted_images/`: Directory containing images from the publication.

## Development
This project uses Astro and React.

To run the site locally:
```bash
cd publication-site
npm install
npm run dev
```
